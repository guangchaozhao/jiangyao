# 官网新闻同步说明

## 当前方案

官网新闻中心不再强依赖 WeWe RSS 导入公众号。

原因：新公众号文章公开链接可以正常访问，但 WeWe RSS 的微信读书解析接口可能返回 `unknown error`，导致公众号导入失败。

现在改为：直接读取公众号文章链接，脚本自动解析文章内容，再生成官网新闻数据。

## 同步内容

脚本会从公众号文章中读取：

- 文章标题
- 公众号名称
- 发布时间
- 正文内容
- 正文图片
- 正文内嵌视频
- 原文链接

正文图片会转存到官网 OSS，避免官网直接加载微信图片导致防盗链或失效。

正文视频如果能在公众号页面中解析到 `mp_video_trans_info`，脚本会选择一份适合网页播放的 mp4 并转存到官网 OSS，避免依赖微信临时签名播放地址。

## OSS 路径

新闻 manifest：

```text
oss://jiangyao-site-2026/news/manifest.json
```

新闻图片：

```text
oss://jiangyao-site-2026/news/images/
```

新闻视频：

```text
oss://jiangyao-site-2026/news/videos/
```

官网读取地址：

```text
https://jiangyao-site-2026.oss-cn-shenzhen.aliyuncs.com/news/manifest.json
```

## 同步命令

单篇文章：

```powershell
npm run sync:wechat-news -- --article-url "https://mp.weixin.qq.com/s/文章ID"
```

多篇文章：

```powershell
npm run sync:wechat-news -- --article-urls "https://mp.weixin.qq.com/s/第一篇,https://mp.weixin.qq.com/s/第二篇"
```

指定文章序号从第 2 篇开始：

```powershell
npm run sync:wechat-news -- --article-url "https://mp.weixin.qq.com/s/文章ID" --article-start 2
```

指定多篇文章序号：

```powershell
npm run sync:wechat-news -- --article-urls "https://mp.weixin.qq.com/s/第一篇,https://mp.weixin.qq.com/s/第二篇" --article-orders "2,3"
```

手动指定分类：

```powershell
npm run sync:wechat-news -- --article-urls "https://mp.weixin.qq.com/s/第一篇,https://mp.weixin.qq.com/s/第二篇" --article-orders "1,2" --article-categories "集团新闻,集团新闻"
```

只本地测试，不上传 OSS：

```powershell
npm run sync:wechat-news -- --article-url "https://mp.weixin.qq.com/s/文章ID" --no-upload
```

## 环境变量

如果不想每次手动传链接，可以在 `.env` 里配置：

```env
WECHAT_NEWS_ARTICLE_URLS=https://mp.weixin.qq.com/s/第一篇,https://mp.weixin.qq.com/s/第二篇
```

上传 OSS 需要：

```env
OSS_ACCESS_KEY_ID=xxx
OSS_ACCESS_KEY_SECRET=xxx
```

也支持项目专用变量：

```env
JIANGYAO_OSS_ACCESS_KEY_ID=xxx
JIANGYAO_OSS_ACCESS_KEY_SECRET=xxx
```

服务器部署时，如果要在服务器跑同步脚本，也要把这些 `.env` 配置同步到服务器。

## 官网展示逻辑

- 新闻中心固定展示三个分类：集团新闻、领导关怀、合作交流。
- 每篇文章在同步时分析一次标题、摘要和正文关键词，然后写入对应分类。
- 分类卡片底部的小框显示该分类下的文章标题。
- 点击文章标题后打开仿公众号阅读排版：标题、公众号名、日期、正文图文、阅读原文。
- 图片和文字不拆开分类，一篇公众号文章始终作为一条图文内容处理。
- 如果公众号正文中有视频，官网阅读页会在原文位置显示可播放视频。

## 当前分类规则

集团新闻：

```text
默认分类。没有明显领导关怀或合作交流关键词时，归入集团新闻。
```

领导关怀：

```text
命中领导、关怀、调研、指导、考察、视察、莅临、慰问、政策等关键词。
```

合作交流：

```text
命中合作、交流、签约、洽谈、共建、入驻、招商、战略、伙伴、校企、赛事机构、媒体等关键词。
```

## 视频说明

公众号文章里的视频优先自动抓取并转存到：

```text
oss://jiangyao-site-2026/news/videos/
```

如果遇到公众号页面没有暴露 mp4 地址，才改为手动上传视频，再在新闻 manifest 里给对应文章补 `videoUrl`。
